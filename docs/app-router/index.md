App Router は、React の最新機能を使用してアプリケーションを構築するための新しい方法です。すでに Next.js に慣れ親しんでいるのであれば、App Router が既存のファイルシステムベースのルーターである[Pages Router](https://nextjs.org/docs/pages)の自然な進化形であることがわかるでしょう。
