# Translation helper

## Install dependencies

```bash
npm i
```

## Set OpenAI API KEY

create `.env.local`:

```bash
export OPENAI_API_KEY="YOUR API KEY HERE"
```

then activate:

```bash
source .env.local
```

## run script

```bash
node translation.js
```

## Warning

- クリップボードへのコピーは Windows 環境で未確認です
